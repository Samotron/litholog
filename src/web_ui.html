About Litholog

                

                    Litholog is a fast, accurate parser for geological descriptions following the BS5930 standard. 
                    It can parse complex soil and rock descriptions and extract structured information.
                

                
                
Supported Features

                
    
                    
  • Material type classification (soil/rock)
    • 
                    
  • Consistency, density, and strength properties
    • 
                    
  • Primary and secondary constituents
    • 
                    
  • Weathering grades and rock structures
    • 
                    
  • Quantitative strength parameters
    • 
                    
  • Intelligent spelling correction
    • 
                    
  • Confidence scoring and validation
    • 
                


                
Performance

                
    
                    
  • Parsing speed: ~1M descriptions/second
    • 
                    
  • Memory efficient: <10MB for typical workloads
    • 
                    
  • Cross-platform: Windows, macOS, Linux
    • 
                    
  • 100% local processing - no data leaves your machine
    •
